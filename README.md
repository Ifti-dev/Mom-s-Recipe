# Mom-s-Recipe
A recipe website where our mothers or anyone can register/login and upload thier food recipe with ingredient list, instruction list which anyone can see and follow the steps to cook food and also give a like.

🌐 Live Site URL
👉 [https://moms-recipe-project.netlify.app/]

Key Features:

✅ Full CRUD Functionality: Create, Read, Update, and Delete recipes.
✅ LocalStorage Integration: All data is handled on the client side—no backend required.
✅ Dynamic UI: Smooth interactions, image uploads, and a personal wishlist.

This project was a great way to master DOM manipulation and browser storage.
